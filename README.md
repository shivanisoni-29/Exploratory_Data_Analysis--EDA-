# Exploratory_Data_Analysis-(EDA)

An interactive **Exploratory Data Analysis (EDA)** project built using **Python, Pandas, Plotly, and Streamlit**. This dashboard analyzes Amazon sales data and provides meaningful business insights through interactive visualizations and dynamic filters.

---

#  Project Overview

The goal of this project is to perform Exploratory Data Analysis (EDA) on Amazon sales data to identify trends, patterns, and key business insights. The processed data is presented through an interactive Streamlit dashboard that allows users to explore sales performance using various filters.

---

#  Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Interactive Streamlit Dashboard
* KPI Cards
* Revenue Analysis
* Monthly Sales Trend
* Category-wise Sales Analysis
* State-wise Revenue Analysis
* Top Cities by Orders
* Order Status Analysis
* B2B vs B2C Analysis
* Executive Business Insights
* Top 5 Revenue Generating Categories
* Download Filtered Dataset
