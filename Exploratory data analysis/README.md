# Amazon Sales Analysis Dashboard

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Amazon sales data and presents insights using an interactive Streamlit dashboard.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Interactive Dashboard
- KPI Cards
- Revenue Analysis
- Monthly Sales Trend
- Top States & Cities
- Order Status Analysis
- Business Insights
- Download Filtered Dataset

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Streamlit

## Run the Project

```bash
streamlit run Dashboard/app.py
```
